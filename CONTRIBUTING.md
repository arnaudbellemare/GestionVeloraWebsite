# Contributing to Gestion Velora Website

Thank you for your interest in contributing.

## How to contribute

This is a private commercial website. External contributions are not accepted at
this time. If you have found a bug or have a suggestion, please contact us at
info@gestionvelora.com.

## Development setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Code style

- TypeScript / React (Vite)
- Follow the existing component structure under `src/`
- Run `npm run build` before submitting any changes to verify the build passes
